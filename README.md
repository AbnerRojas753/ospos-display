# OSPOS Display

Muestra la IP local y la URL del sistema OSPOS en una ventana.

## Requisitos

- Python 3.x instalado en Windows

## Uso

1. Copia la carpeta `ospos-display` a tu máquina Windows
2. Ejecuta `run.bat` o:
   ```
   python ospos_display.py
   ```

## Captura

El programa muestra:
- **IP Local**: La dirección IP de la máquina (ej. 192.168.2.100)
- **URL OSPOS**: La URL completa para acceder al sistema (ej. http://192.168.2.100:8000)
- Botón para copiar la URL al portapapeles
